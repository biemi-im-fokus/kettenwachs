📖 Anleitung

Grundidee: Du hast zwei Ketten, aber immer nur eine ist montiert. Gefahrene Kilometer zählen automatisch auf die montierte Kette. Wachsen setzt deren Zähler zurück.

🚴 Fahrt eintragen

Datum (Standard heute), Kilometer und optional eine Notiz eingeben, dann ＋. Die km werden der aktuell montierten Kette gutgeschrieben — darunter steht, welche das ist. Enter im km-Feld speichert ebenfalls.

🔗 Die zwei Ketten-Karten

  • Großer Ring = km seit der letzten Auffrischung (💧) gegen das Flüssigwachs-Intervall.
  • Balken darunter = 🔥 Heißwachs-Basis: km seit dem letzten Heißwachs gegen das lange Intervall.
  • Farben: grün = alles gut, gelb = bald fällig (ab 80 %), rot = überfällig.
  • Status-Pille zeigt die dringendere der beiden Aktionen an.
  • Gesamt = Laufleistung der Kette (inkl. Start-km), darunter die Daten des letzten 🔥 und 💧.

🔥 Heißwachs & 💧 Flüssigwachs

  • 🔥 Heißwachs = die Basis (langes Intervall). Setzt beide Zähler zurück.
  • 💧 Flüssig = schnelles Nachbessern (kurzes Intervall). Setzt nur den Auffrisch-Zähler zurück — die Basis altert weiter.
  • Beim Tippen öffnet sich ein Datums-Picker plus optionales Notizfeld.

🔄 Kette montieren / wechseln

Tippst du bei der nicht-montierten Kette auf „↑ Diese Kette montieren", zählen ab diesem Tag die Fahrten auf diese Kette. So bildest du den realen Ketten-Tausch am Rad ab. Die montierte Kette trägt das Abzeichen „montiert".

🔔 Erinnerung

Oben erscheint ein Banner, sobald die montierte Kette fällig wird — gelb („bald") oder rot („fällig"), mit Hinweis ob 🔥 oder 💧 dran ist. (Echte Push-Nachrichten sind in einer Browser-Datei nicht möglich — die Erinnerung ist sichtbar, sobald du das Tool öffnest.)

📜 Verlauf

  • Nach Monaten gruppiert und einklappbar; neuester Monat offen. Rechts pro Monat eine Mini-Bilanz (Fahrten · km).
  • Zeile antippen oder ✎ öffnet die Bearbeitung: Datum, km / Wachs-Art / Kette und Notiz ändern.
  • ✕ löscht einen Eintrag (mit Rückfrage). Löschen geht auch im Bearbeiten-Fenster.

⚙︎ Einstellungen

  • Intervalle für Heißwachs & Flüssigwachs (km bis zur Erinnerung).
  • Namen der beiden Ketten (auch per Klick auf den Namen änderbar).
  • Start-km = Laufleistung, die eine Kette schon vor dem Tool hatte (zählt nur zur Gesamt-Laufleistung).

💾 Daten & Backup

Alle Daten liegen lokal im Browser (localStorage) — offline-fähig und schnell, aber an diesen Browser gebunden. Über Export (JSON) sicherst du alles als Datei (Backup oder zum Weitergeben), mit Import lädst du sie wieder ein.

Einstellungen

Heißwachs = Basis mit langem Intervall. Flüssigwachs = schnelles Nachbessern dazwischen mit kürzerem Intervall. Ein Heißwachs setzt beide Zähler zurück, ein Flüssigwachs nur den Nachwachs-Zähler.

Start-km = Laufleistung, die eine Kette schon hatte, bevor du das Tool nutzt. Zählt nur zur Gesamt-Laufleistung.

Export lädt eine JSON-Datei mit allen Fahrten & Einstellungen herunter — als Backup oder zum Weitergeben. Import liest so eine Datei wieder ein und ersetzt die aktuellen Daten.
